Product: Cyclomat Website (Marketing + Learning Experience)
1. Product Overview
Product Name
Cyclomat Website
Product Type
Marketing + onboarding + educational website
Objective
Create a visually compelling, intuitive website that:
1.	Explains what Cyclomat is
2.	Demonstrates what it creates
3.	Teaches users how to use it
4.	Builds confidence for first-time users
This site acts as the front door + onboarding layer for Cyclomat.
2. Background & Context
Cyclomat is an application that generates intricate geometric artwork through:
·	Harmonic motion
·	Rotating “arms”
·	Frequency relationships
·	Layered compositions
·	Transformations and keyframes
While powerful, the tool introduces unfamiliar concepts (frequency, phase, oscillation), which can create friction for new users.
3. Problem Statement
New users currently face:
·	Unclear understanding of what Cyclomat does
·	Difficulty mapping controls to outcomes
·	Overwhelm from advanced parameters
·	Lack of structured learning path
Without guidance:
·	Users experiment randomly
·	Fail to produce satisfying results early
·	Lose interest before understanding the system

9.2 How It Works — Conceptual Model




Purpose
Explain core system without UI complexity
Concepts
Cycloid
A curve generated by rotating motion
Arms
Independent rotating contributors to the shape
Frequency
Controls how many rotations occur → defines shape family
Radius
Controls influence of each arm → defines variation within family
Phase
Controls starting angle → changes structure dramatically
Key Principle
Frequency defines the structure. Radius defines the expression.
